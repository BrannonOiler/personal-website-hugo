# Personal Website

This is my personal website built using Hugo with the Hugo Coder theme. It contains an about me section and a blog where I can post projects and other random thoughts.

## Getting Started

To run the website locally, follow these steps:

1. Clone the repo and start the Hugo server (must have Hugo installed):

```bash
git clone https://github.com/brannonoiler/personal-website-hugo.git
cd personal-website-hugo && hugo server -D
```

2. Open your browser and go to `http://localhost:1313` to see the website.
